## Initiate project 👨‍💻

To run the project, simply plit the terminal and type the following in seperate terminals:

#Terminal 1

## Install and Startup

Navigate to server

```
cd server
```

Install packages

```
npm install
```

Start server

```
npm start
```

---

#Terminal 2

## Install and Startup

Navigate to Client (React)

```
cd client
```

Install packages

```
npm install
```

Start project

```
npm start
```
